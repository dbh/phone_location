# phone_location

## Intro 
Phone Location is an opportunity to play with Mobile Phone application development, Geo Location, MQTT Messaging, persistence, and map-based visualization.

## Tech used 
so far...
* Flutter / Dart
* Geo Location
* MQTT
* Python
* Maria DB
